# Exercise 1 - Basic Spring Boot Application

## Objective

Create a Spring Boot Web Project using Maven and understand the basic project structure.

## Technologies Used

- Java 17
- Spring Boot 3.5.x
- Maven
- Spring Web
- Spring Boot DevTools

## Project Structure

- src/main/java - Application source code
- src/main/resources - Configuration files
- src/test/java - Test classes
- pom.xml - Maven configuration

## Execution

Run the `SpringLearnApplication` class.

Application starts successfully on the embedded Tomcat server.

## Output

```
Tomcat started on port(s): 8080 (http)
Started SpringLearnApplication
```

## Conclusion

Successfully created and executed a Spring Boot application using Maven.
