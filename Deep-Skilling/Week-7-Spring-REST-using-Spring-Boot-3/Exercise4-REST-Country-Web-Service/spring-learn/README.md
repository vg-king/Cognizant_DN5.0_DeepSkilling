# Exercise 4 - REST Country Web Service

## Objective

Create a REST API that returns Country details loaded from Spring XML configuration.

## Technologies

- Java 17
- Spring Boot
- Spring Web
- Spring Core XML

## Endpoint

GET /country

## URL

http://localhost:8083/country

## Response

```json
{
  "code": "IN",
  "name": "India"
}
```

## Conclusion

Successfully created a REST API that loads a Spring Bean from XML configuration and returns it as JSON.
