# Exercise 4: Creating and Configuring a Maven Project

## Objective

The objective of this exercise is to create a Maven-based Spring project and configure the required Spring dependencies and Maven Compiler Plugin.

---

## Technologies Used

- Java 17
- Maven
- Spring Context
- Spring AOP
- Spring Web MVC
- IntelliJ IDEA

---

## Dependencies

- Spring Context
- Spring AOP
- Spring WebMVC

---

## Maven Plugin

The Maven Compiler Plugin is configured to compile the project with Java 1.8 source and target compatibility.

---

## Key Concepts Learned

- Maven Project Structure
- Dependency Management
- Maven Compiler Plugin
- Spring Dependencies
- Build Configuration

---

## Conclusion

This exercise demonstrates how Maven manages project dependencies and plugins, making Java application development easier and more maintainable.
