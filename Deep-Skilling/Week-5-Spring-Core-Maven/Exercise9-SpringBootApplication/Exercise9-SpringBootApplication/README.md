# Exercise 9: Creating a Spring Boot Application

## Objective

The objective of this exercise is to build a Spring Boot application for a Library Management System using Spring Web, Spring Data JPA, and the H2 in-memory database. The application demonstrates RESTful CRUD operations using Spring Boot.

---

## Technologies Used

- Java 17
- Spring Boot
- Spring Web
- Spring Data JPA
- H2 Database
- Maven
- IntelliJ IDEA

---

## Features

- Spring Boot Project
- REST API
- Spring Data JPA Repository
- H2 In-Memory Database
- CRUD Operations
- Auto Configuration

---

## REST Endpoints

| Method | Endpoint | Description |
|---------|----------|-------------|
| GET | /books | Get all books |
| POST | /books | Add a new book |

---

## Key Concepts Learned

- Spring Boot
- Auto Configuration
- REST Controller
- Spring Data JPA
- H2 Database
- Entity
- Repository
- CRUD Operations

---

## Conclusion

This exercise demonstrates how Spring Boot simplifies application development by providing auto-configuration, embedded server support, and seamless integration with Spring Data JPA and H2 Database.
